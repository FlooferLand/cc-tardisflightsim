Welcome to your personal T.A.R.D.I.S training experience!
The settings will be presented in this folder as "conf.json"

# Redstone
The program can accept redstone signals from one side in order to handle events
View eventTraining.lua to see which redstone signal corresponds to what temporal event
It can also accept a throttle value (made to be used with Create's analogue lever)

# Themes
The software currently has 2 themes:
 - "generic" (default)
 - "steampunk"
